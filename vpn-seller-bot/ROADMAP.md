# VPN Project Roadmap — @Poliakarbot

> Сервер: 2.27.48.142 (Хельсинки, Финляндия)
> Обновлён: 15.06.2026

## ✅ Level 0–2 — Сделано

- [x] VLESS+REALITY основной (4443) + резервный (8445), разные камуфляжи
- [x] WireGuard (443/UDP) + udp2raw TCP-обфускация (51820/TCP)
- [x] MTProto прокси (8443)
- [x] DDoS-защита: iptables hashlimit на всех портах
- [x] Авто-ротация камуфляжных доменов (12 доменов, каждые 6ч)
- [x] Мониторинг: vpn-watch v2.3 (сервис, порт, трафик, conntrack, idle)
- [x] TCP-тюнинг: 16MB буферы + BBR + MSS clamp
- [x] Mux (concurrency=8): коннекты с 10K до ~100
- [x] Conntrack overflow detector (авто-рестарт Xray при >500 мёртвых сессий)
- [x] Telegram-бот @Poliakarbot с продажами через CryptoBot (TON/USDT)
- [x] Админ-панель: /admin stats, /admin traffic, /admin health, /admin issue
- [x] Бэкапы: БД ежедневно, снапшоты конфигов, 7-дневная ротация
- [x] Цены: VLESS 30д/$2, 90д/$5.4, WG 30д/$1, MTProto 30д/$0.5

## 🔜 Level 3 — Анти-цензура 2.0

- [ ] **Hysteria2** — протокол на QUIC поверх UDP, быстрее VLESS, сложнее блокировать DPI
  - Порт: 8443/UDP (рядом с MTProto)
  - Клиенты: Hysteria Android/iOS, sing-box, NekoBox
  - Интеграция в бота: новый тип подписки «VLESS+Hysteria»
- [ ] **Sing-box** — ядро вместо/вместе с Xray
  - Лучше маскировка под HTTP/3
  - Одно ядро для VLESS + Hysteria2 + TUIC
- [ ] **Авто-смена IP при блокировке**
  - Мониторинг доступности из РФ (внешний чекер)
  - Запасной VPS с быстрым переключением DNS/IP
- [ ] **Multi-hop** — цепочка из 2+ серверов
  - Входной (Финляндия) → выходной (Нидерланды)
  - Сложнее отследить конечный трафик

## 📈 Level 4 — Рост продаж

- [ ] **Триал 24 часа** — бесплатно, привязка к телефону/email
  - Ограничение скорости на триале (5 Mbps)
  - Авто-отключение через 24ч, предложение купить
- [ ] **Реферальная система** — «приведи друга — +7 дней обоим»
  - Уникальная реферальная ссылка на каждого пользователя
  - Отслеживание в БД: referrer_id, бонус начислен
- [ ] **Лендинг / мини-сайт** — продажа без Telegram
  - Статический HTML + авто-выдача через API
  - SEO, таргетированная реклама
- [ ] **Пакетные скидки**
  - 3 месяца: −10%, 6 месяцев: −20%, год: −35%
  - Групповые/семейные тарифы (до 5 устройств)
- [ ] **Промо-каналы**
  - Авто-постинг в тематические Telegram-каналы
  - Партнёрства с блогерами
